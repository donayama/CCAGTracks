Concept

CCAGは「架空のゲーム」と「そのオリジナルサウンドトラック」を セットで制作・公開するシリーズです。各#NNは、 ゲーム設定・ライナーノーツ・生成レシピ（jobs.toml）・音源を ひとつの作品単位として管理しています。

  • 全作品はフィクションを前提にした創作プロジェクトです。
  • 実在のゲーム・企業・団体・人物とは関係ありません。
  • 音楽はACE-Stepベースのワークフローで生成・編集しています。
  • 使用規約・ライセンス運用は README を参照してください。
